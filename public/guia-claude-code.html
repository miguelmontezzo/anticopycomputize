01

O que é o Claude Code?

Entenda em 2 minutos

Você já deve ter ouvido falar do Claude — aquele assistente de IA que responde perguntas, escreve textos e ajuda a pensar. O Claude Code é uma versão especial do Claude que não fica só falando — ele age.

Claude Normal
É como perguntar para um amigo especialista. Ele te diz o que fazer, mas você que tem que fazer.
Claude Code
É como ter um assistente que faz por você. Você pede, ele executa.

O Claude Code tem "mãos". Isso quer dizer que ele consegue:

📁
Ler e criar arquivos
Cria documentos, planilhas, código e organiza pastas no seu computador — sem você precisar abrir nenhum app.
⚡
Rodar programas
Executa scripts, instala ferramentas e testa se tudo está funcionando corretamente.
🔍
Pesquisar na internet
Busca informações atuais quando precisa resolver problemas ou trazer dados recentes.
🔗
Conectar com outros serviços
Pode acessar GitHub, Notion, Gmail e outros apps via integrações — tudo automaticamente.
// A grande diferença
Um chatbot responde. O Claude Code age. Ele não só te diz "crie um arquivo assim" — ele cria o arquivo, testa, corrige os erros e avisa quando terminou. É a diferença entre um consultor e um assistente que coloca a mão na massa.
02

Como funciona na prática?

O loop de raciocínio do agente

Quando você dá uma tarefa para o Claude Code, ele não vai direto fazer. Ele pensa primeiro, divide em partes menores, e executa passo a passo — como um funcionário organizado.

1
Perceber
Lê seus arquivos, entende o projeto e identifica o que precisa ser feito
2
Planejar
Divide a tarefa em etapas e decide quais ferramentas vai usar
3
Agir
Executa: cria arquivos, roda comandos, busca informações
4
Verificar
Checa se deu certo, corrige erros e ajusta o que for necessário
5
Repetir
Volta ao início até completar a tarefa ou pedir ajuda

Por onde começar?

Roteiro passo a passo — primeiros 7 dias

Siga este roteiro progressivo. Em 7 dias você vai ter uma visão clara de como o Claude Code pode mudar sua forma de trabalhar.

Dia 1
Explore o Claude no navegador
  • Abra claude.ai no navegador e crie uma conta gratuita
  • Converse com o Claude: peça para ele explicar algo ou escrever um texto
  • Objetivo: sentir como é conversar com uma IA de verdade
🌐 Acesse claude.ai → clique em "Sign up" → crie sua conta com e-mail ou Google. É grátis.
Dia 2–3
Instale o Claude Code
  • Assine o plano Pro em claude.ai/settings
  • Abra o terminal do seu computador
  • Cole o comando de instalação abaixo
💻
Como abrir o Terminal:
→ Mac: pressione Cmd ⌘ + Espaço, digite "Terminal", pressione Enter
→ Windows: pressione Win + R, digite "cmd", pressione Enter
📋 Com o Terminal aberto, copie e cole o comando abaixo. Depois pressione Enter:
Terminal
$ npm install -g @anthropic-ai/claude-code
# Instala o Claude Code no seu computador
added 1 package in 3s
✓ claude-code instalado com sucesso
📂 Agora, no Terminal, navegue até qualquer pasta do seu computador e inicie o agente:
Terminal
$ claude
# Inicia o Claude Code na pasta atual
✓ Claude Code iniciado
→ Como posso ajudar?
// Primeira coisa a pedir
Digite: "O que tem nessa pasta? Me explica o que é esse projeto." — ele vai ler os arquivos e te dar um resumo automaticamente.
Dia 4–5
Crie seu primeiro projeto real
  • Escolha algo pequeno: um site simples, uma planilha, ou um script pra organizar arquivos
  • Descreva o que você quer com detalhes — como se fosse explicar para um funcionário novo
  • Deixe o Claude Code trabalhar e apenas observe o processo
💡
Exemplo de pedido que funciona muito bem:

"Cria um site de portfólio simples para mim com meu nome, minhas habilidades e um formulário de contato. Usa HTML, CSS e JavaScript puro."
// Dica de ouro
Quanto mais detalhes você der, melhor o resultado. Pense em descrever o que você quer como se fosse para um funcionário que nunca te viu: quem você é, o que precisa, como deve ficar.
Dia 6–7
Crie o arquivo CLAUDE.md
  • Crie um arquivo chamado CLAUDE.md na raiz do seu projeto
  • Escreva nele: quem você é, o que o projeto faz, como o agente deve se comportar
  • Teste tarefas mais longas e observe a diferença
📝 O CLAUDE.md é o manual de instruções permanente para o agente. Tudo que você escrever lá, ele vai seguir em todas as sessões — sem você precisar repetir.

// Mac — crie e abra o arquivo

Terminal (Mac)
$ touch CLAUDE.md && open CLAUDE.md
# Cria o arquivo e abre para edição

// Windows — crie e abra o arquivo

Terminal (Windows)
> echo. > CLAUDE.md && notepad CLAUDE.md
# Cria o arquivo e abre no Bloco de Notas
// Próximo passo: integrações
Depois de dominar o básico, conecte com GitHub, Notion ou outras ferramentas. O Claude Code pode criar branches, abrir pull requests e muito mais — automaticamente.
